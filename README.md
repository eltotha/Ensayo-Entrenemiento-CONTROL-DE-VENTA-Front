# Control de Venta

Proyecto frontend de control de ventas usando React y Vite.

## Requisitos

- Node.js instalado
- npm disponible en la terminal

## Instalación

Desde la carpeta raíz del proyecto, ejecuta:

```bash
npm install
```

Esto descargará e instalará todas las dependencias listadas en `package.json`.

## Ejecución en desarrollo

Para iniciar el servidor de desarrollo con recarga automática:

```bash
npm run dev
```

Después de ejecutar este comando, abre la URL que muestre Vite en la terminal (en nuestro caso `http://localhost:5000`).

## Compilar para producción

Para generar los archivos optimizados de producción:

```bash
npm run build
```

Los archivos generados se ubicarán en la carpeta `dist`.

## Vista previa de producción

Para ver el sitio generado en modo producción localmente:

```bash
npm run preview
```

## Notas

- `node_modules` no se incluye en el repositorio.
- Si ya has clonado el proyecto, basta con ejecutar `npm install` y luego `npm run dev`.
